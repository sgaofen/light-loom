# 织光 · Light-Loom

> 一首为八个声部而织的电子复调 · a polyphony in eight voices
> 112 BPM · D Dorian → E Minor · 3′17″

一个**单文件、零依赖、完全离线**的原创音乐可视化作品。整首曲子的乐谱与全部音色都由浏览器用 Web Audio **实时合成**，画面由 Canvas **实时绘制**——没有任何外部音频、图片或字体文件。

**▶ 在线播放：** https://stephenyuuu.github.io/light-loom/

或下载 [`index.html`](index.html) 双击用 Chrome / Edge / Safari 打开即可。

## 玩法

- 圆环音游式可视化：音符从外圈坠向判定环，命中爆出光屑；每个声部一条弧形轨道随其实时音量发光
- **点右侧任意声部可单独静音**——试试只留主旋律和钟琴，是另一首曲子
- 空格暂停 / 继续，点进度条跳转，右下角可导出无损 WAV

## 结构

九个段落：**启 · 纺 · 织 · 流 · 隐 · 潮 · 暗涌 · 焰 · 归**。
D 多利亚出发，终段升全音入 E 小调。0:34 主旋律登场，1:46 第一次全员落下，2:17 升调后的最终高潮。

八个声部 + 氛围层，全部 Web Audio 合成：FM 钟琴、失谐锯齿云幕、3 对 4 交叉律动的纺线琶音、桥段藏的 7/16 复合节拍、侧链泵吸、卷积混响 + 乒乓延迟。
